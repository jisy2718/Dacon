### Loss

+ Log Loss 이용
    $$
    Log Loss = - (1/n) * Σ_{i=1}^n (y_i * log(p_i) + (1 - y_i) * log(1 - p_i))
    $$

+ 이진 분류에서 사용되는 Loss로, 실제 $y$ 값과 $y$ 에 대한 예측 확률인 $p$ 값이 일치할수록 Loss가 작아진다.

    + $y \in \{0,1\}$

    + $p_i$ 는 $y=1$ 일 확률
